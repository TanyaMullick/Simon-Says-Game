# 🎮 Simon Says Game

An interactive memory-based game inspired by the classic *Simon Says*, developed using **HTML, CSS, and JavaScript**. The application dynamically generates color sequences that the user must replicate, with increasing difficulty at each level.

---

## 📌 Project Overview

This project demonstrates core **front-end development skills**, including DOM manipulation, event handling, game logic implementation, and responsive UI design. The game progressively increases complexity, enhancing user engagement and logical flow control.

---

## 🚀 Key Features

* Dynamic random sequence generation
* Level-based difficulty progression
* Real-time user input validation
* Visual feedback for interactions
* Game-over detection with restart functionality

---

## 🛠️ Tech Stack

* **HTML5** – Structure and layout
* **CSS3** – Styling and responsive design
* **JavaScript (Vanilla JS)** – Game logic, DOM manipulation, event handling

---

## 🎯 How It Works

1. The game starts on key press.
2. A random color flashes as part of the sequence.
3. The player repeats the pattern by clicking the buttons.
4. With each correct round, a new color is added to the sequence.
5. The game ends when the player selects the wrong color.
